A - событие попадание дротиком в мишень
P - вероятность

P(A) - вероятность попадания дротиком в мишень
P(B) - вероятность попадания дротиком мимо мишени

Расчет вероятности относительно площади:

Sa - площадь мишени
Sb - оставшаяся площадь
Sw - вся площадь стены

P(A) = Sa / Sw = 0 / 100 = 0        P(B) = Sb / Sw = 100 / 100 = 1

P(A) + P(B) = 1

P(A) = Sa / Sw = 50 / 100 = 0.5     P(A) = Sb / Sw = 50 / 100 = 0.5

Расчет вероятности относительно колличества бросков:

Na - сколько раз была порожена цель
Nw - общее колличество бросков

P(A) = Na / Nw = 0 / 100 = 0

P(A) = Na / Nw = 50 / 100 = 0.5

Вероятность какого-либо события равна от 0 до 1 0 <= P <= 1

Грани купика пронумерованы от 1 до 6

A1, A2, A3, A4, A5, A6 - полная группа событий (какая-либо выпадет в при любом исходе)
 размер этой полной группы равен n = 6 и m = 1 - это вероятность выпадения какой-либо из граней кубика

Вероятность появления каждого события = 1/6
P(A1) = P(A2) = P(A3) = P(A4) = P(A5) = P(A6) = 1 / 6

P(A1) + P(A2) + P(A3) + P(A4) + P(A5) + P(A6) = 1

если мы хотим посчитать вероятность выпадения A1 или A5 то n = 6, а вот m = 2 - колличество благоприятных исходов

P(A) = m/n = 2/6 = 1/3

В фирме такси в данный момент свободно 35 машин: 11 красных, 17 фиолетовых и 7 зеленых.
По вызову выехала одна из машин, случайно оказавшаяся ближе всего к заказчице.
Найдите вероятность того, что к ней приедет зеленое

n = 35
m = 7 - так как благоприятных исходов у нас 7
P(A) = m/n = 7/35 = 1/5 = 0.2
